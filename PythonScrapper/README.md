# PythonScrapper

## Description
This is an API which returns data from bookmakers, especially betting ods.

## Installation

### Manual

- Fill the env variables mentionned in .env.example

- At the root of the repo:
```
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

### Docker

```
docker compose up
```
